# ndarrayオブジェクトの保存

numpyオブジェクトの保存と読み込みを行う

Sample

```python
# coding:utf-8
import numpy as np

# 標準正規分布による乱数を値に持つ3x3行列
x = np.random.randn(3, 3)
print x

# xを保存する
np.save('random.npy', x)

# xを読み込む
y = np.load('random.npy')
print y
```
