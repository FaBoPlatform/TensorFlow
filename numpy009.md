# ブールインデックス

Sample

```python
# coding:utf-8
import numpy as np

# 標準正規分布による乱数を10コ生成する
x = np.random.randn(10)
print x

# 0以上の場合にはTrueを返す
mask = x >= 0.0
print mask

# Trueとなっている値のみ取得する
print x[mask]
# 以下も同様
print x[x >= 0]
```
