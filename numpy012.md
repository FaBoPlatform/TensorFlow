# CSVファイルの読み込み・保存

CSV(コンマ区切り`,`)およびTSV(タブ区切り`\t`)から値を読み込む。

## Sample

```python
# coding:utf-8
import numpy as np

# CSVファイルの読み込み
x = np.loadtxt('hoge.csv', delimiter=',')
print x
# TSVファイルの読み込み
y = np.loadtxt('piyo.tsv', delimiter='\t')
print y

# CSVファイルの保存
np.savetxt('foo.csv', x, delimiter=',')
```

`hoge.csv` : 

```
0.2,3.398,592
1.3,0.83,8.8
```

tsvのサンプルはコピーだと、Tabが無効になるので、Tabを自分でうつようにする

`piyo.tsv` : 

```
0.2	3.398	592
1.3	0.83	8.8
```
